**Ageing** 是一个以年龄为主题的简单游戏内玩家评分系统，主要对外提供 API 由其它插件调用。Ageing 可以用来控制玩家在指定的等级能够做什么事。详细介绍内容和文档见 https://book.sotap.org/#/Ageing/index 。
